# Notes Management

Notes store reusable content captured from chat, summaries, or manual entry.

---

## What It Does

- Save chat responses as reusable notes
- Create manual notes from scratch
- Convert notes into sources for future queries
- Organize, rename, download, and delete notes

---

## Why It Matters

Notes preserve key findings and allow reusing them as sources. This builds a personal knowledge base within HALO Core.

---

## Creating Notes

### From Chat Response

1. Below a chat response, click **In Notiz speichern**
2. Note is saved with the response content
3. Source info is preserved

### From Source Summary

1. In the summary section, click save
2. Summary becomes a note
3. Can be used as a source

### Manual Note

1. In notes section, click **Notiz hinzufügen**
2. Enter title and content
3. Save

---

## Viewing Notes

### Expand a Note

1. Click on a note to expand
2. View content and source info
3. See creation timestamp

### Note Details

| Field | Description |
|-------|-------------|
| Title | Note name |
| Content | Saved text |
| Source | Origin (chat, summary, manual) |
| Date | Creation timestamp |

---

## Managing Notes

### Rename

1. Click the note menu (⋮)
2. Select **Rename**
3. Enter new name
4. Confirm

### Download

1. Click the note menu (⋮)
2. Select **Download**
3. File downloads to your computer

### Convert to Source

1. Click the note menu (⋮)
2. Select **Convert to Source**
3. Note appears in Sources list
4. Can be selected for chat/studio

### Delete

1. Click the note menu (⋮)
2. Select **Delete**
3. Confirm deletion

!!! warning
    Deleting a note removes it permanently. This cannot be undone.

---

## Notes vs Sources

| Aspect | Notes | Sources |
|--------|-------|---------|
| Origin | Created in HALO | Imported/uploaded |
| Purpose | Save insights | Input for AI |
| Conversion | Can become sources | Cannot become notes |
| Editing | Can be edited | Cannot be edited |

---

## Best Practices

### What to Save as Notes

- Key findings from chat
- Important summaries
- Decisions and conclusions
- Reusable explanations

### What Not to Save

- Temporary queries
- Incomplete responses
- Content you'll delete soon

### Organization

- Use descriptive titles
- Convert important notes to sources
- Delete outdated notes regularly

---

## Troubleshooting

### "Note doesn't appear after saving"

Try this:

1. Refresh the notes section
2. Check for error messages
3. Verify data directory is writable

### "Convert to source fails"

Try this:

1. Ensure note has content
2. Check for special characters in title
3. Verify source list refreshes

# Studio

The Studio panel is where you generate structured deliverables from your current context using template cards.

---

## What It Does

- Generate reports, infographics, podcasts, presentations, and data tables
- Customize output language, tone, and focus
- Store and export generated artifacts
- Link outputs to source context

---

## Why It Matters

Studio transforms chat/research context into actionable artifacts quickly. Instead of manually formatting outputs, you use templates designed for specific deliverable types.

---

## Available Templates

| Template | Description |
|----------|-------------|
| **Bericht** (Report) | Structured written report |
| **Infografik** | Visual infographic outline |
| **Podcast** | Podcast script and structure |
| **Videoubersicht** | Video script and storyboard |
| **Präsentation** | Slide deck outline |
| **Datentabelle** | Structured data table |

---

## Using Templates

### Basic Flow

1. Open **Studio** panel (right column)
2. Pick a template card
3. Click the generate button
4. Review saved output in **Studio-Ergebnisse**

### Customizing Output

Before generating, you can customize:

1. Click the card menu (⋮)
2. Set options:
   - **Language**: Output language
   - **Tone**: Writing style
   - **Extra prompt**: Additional instructions
3. Generate

---

## Expected Results

After generation:

- New output artifact appears in Studio results
- Output includes generation timestamp
- Source context is linked
- You can expand, review, export, or delete

---

## Managing Outputs

### View Output

1. In **Studio-Ergebnisse**, find the output
2. Click to expand and read content

### Export Output

1. Click the output menu (⋮)
2. Select **Download**
3. Choose format (if applicable)

### Delete Output

1. Click the output menu (⋮)
2. Select **Delete**
3. Confirm deletion

---

## Output Details

Each output includes:

| Field | Description |
|-------|-------------|
| Title | Generated or custom title |
| Type | Template used |
| Timestamp | When generated |
| Sources | Linked source context |
| Content | Generated text |

## Best Practices

### Before Generation

- Ensure relevant sources are selected
- Review chat context for key findings
- Set appropriate language and tone

### After Generation

- Review output for accuracy
- Edit or refine as needed
- Export for external use

### For Reports

- Use Bericht template for comprehensive documents
- Include extra prompt for specific focus areas

### For Presentations

- Use Präsentation template for slide outlines
- Export and refine in presentation software

---

## Troubleshooting

### "No studio outputs generated"

Possible causes:

- No relevant selected source context
- Model/tool configuration issue
- API connectivity problem

Try this:

1. Confirm source selection
2. Test with Bericht template first
3. Check chat and agent configuration

### "Output quality is poor"

Possible causes:

- Insufficient source context
- Vague extra prompt
- Wrong tone setting

Try this:

1. Add more relevant sources
2. Provide specific extra prompt instructions
3. Adjust tone setting

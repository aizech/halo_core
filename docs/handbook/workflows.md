# Workflows

This section provides step-by-step guides for common tasks.

---

## Workflow 1: Ask Grounded Questions on Uploaded Documents

### Goal

Get traceable answers tied to your source material.

### Steps

1. **Upload documents**
   - In Sources, click **+ Quellen hinzufügen**
   - Select PDFs, DOCX, or other files
   - Wait for ingestion

2. **Select sources**
   - Mark uploaded files with checkboxes
   - Ensure at least one source is selected

3. **Ask a focused question**
   - In Chat, enter your question
   - Be specific about what you need

4. **Review the answer**
   - Check source citations
   - Expand Agent Actions if needed

5. **Save the best answer**
   - Click **In Notiz speichern**
   - Note appears in Notes section

### Result

You have a traceable answer tied to selected material, saved for future reference.

## Workflow 2: Build a Report Draft from Collected Evidence

### Goal

Create a structured report from multiple sources.

### Steps

1. **Import sources**
   - Upload relevant files
   - Use connectors for external content
   - Select all relevant sources

2. **Generate summary**
   - In Chat, open **Zusammenfassung aller Quellen**
   - Click refresh to generate
   - Review the summary

3. **Open Studio**
   - Navigate to Studio panel
   - Select **Bericht** template

4. **Customize output**
   - Open template menu
   - Set language and tone
   - Add extra prompt with focus areas

5. **Generate and export**
   - Click generate button
   - Review output in Studio-Ergebnisse
   - Download for external editing

### Result

Report-style draft you can refine in external tools.

## Workflow 3: Convert Insights into Presentation-Ready Structure

### Goal

Create a slide deck outline from your research.

### Steps

1. **Prepare source set**
   - Select relevant documents
   - Ensure key findings are captured

2. **Chat for key points**
   - Ask for main themes
   - Request bullet points
   - Save important responses as notes

3. **Use Präsentation template**
   - Open Studio
   - Select **Präsentation** template
   - Set language and style

4. **Generate output**
   - Click generate
   - Review slide outline

5. **Export and refine**
   - Download the output
   - Transfer to presentation software
   - Add visuals and formatting

### Result

Rapid first draft of slide flow ready for refinement.

## Workflow 4: Manage Long-Running Project Memory

### Goal

Keep assistant behavior clean and relevant over time.

### Steps

1. **Review memory entries**
   - Open Account page
   - View stored memory items

2. **Delete outdated entries**
   - Select irrelevant memories
   - Click delete selected
   - Or clear all if needed

3. **Keep useful context**
   - Leave entries that help
   - Remove noise and outdated info

### Result

Cleaner assistant behavior and reduced noise in responses.

## Workflow 5: Multimodal Analysis

### Goal

Combine text, images, and audio in one analysis.

### Steps

1. **Upload image**
   - In Chat, click image icon
   - Select image file

2. **Add audio context**
   - Click audio icon
   - Record or upload audio
   - Review transcription

3. **Enter prompt**
   - Ask about both image and audio content
   - Request specific analysis

4. **Review response**
   - Check image analysis
   - Verify audio context integration

5. **Save result**
   - Save as note for reference

### Result

Comprehensive analysis combining multiple input types.

## Workflow 6: Quick Research Synthesis

### Goal

Rapidly synthesize information from multiple sources.

### Steps

1. **Select diverse sources**
   - Choose documents from different perspectives
   - Include primary and secondary sources

2. **Generate all-sources summary**
   - Use summary feature
   - Review key themes

3. **Ask synthesis question**
   - Request comparison or synthesis
   - Ask for conflicting viewpoints

4. **Generate infographic**
   - Use **Infografik** template
   - Visualize key findings

### Result

Quick synthesis with visual output for sharing.

---

## Workflow 7: Podcast Script Creation

### Goal

Create a podcast script from research material.

### Steps

1. **Prepare content**
   - Select source documents
   - Generate summary for context

2. **Open Studio**
   - Select **Podcast** template
   - Set language and tone

3. **Customize for your format**
   - Add extra prompt with:
     - Episode length
     - Target audience
     - Key topics to cover

4. **Generate and review**
   - Create script
   - Edit as needed

### Result

Podcast script ready for recording.

## Tips for All Workflows

### Before Starting

- Clear irrelevant selections
- Verify API connectivity
- Check source quality

### During Work

- Save important outputs immediately
- Use notes for intermediate results
- Check citations for accuracy

### After Completion

- Export final outputs
- Clean up temporary notes
- Document your process

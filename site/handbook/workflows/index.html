    Workflows

    This section provides step-by-step guides for common tasks.

    Workflow 1: Ask Grounded Questions on Uploaded Documents

    Goal

    Get traceable answers tied to your source material.

    Steps

    1. Upload documents
    2. In Sources, click + Quellen hinzufügen
    3. Select PDFs, DOCX, or other files

    4. Wait for ingestion

    5. Select sources

    6. Mark uploaded files with checkboxes

    7. Ensure at least one source is selected

    8. Ask a focused question

    9. In Chat, enter your question

    10. Be specific about what you need

    11. Review the answer

    12. Check source citations

    13. Expand Agent Actions if needed

    14. Save the best answer

    15. Click In Notiz speichern
    16. Note appears in Notes section

    Result

    You have a traceable answer tied to selected material, saved for future reference.

    Workflow 2: Build a Report Draft from Collected Evidence

    Goal

    Create a structured report from multiple sources.

    Steps

    1. Import sources
    2. Upload relevant files
    3. Use connectors for external content

    4. Select all relevant sources

    5. Generate summary

    6. In Chat, open Zusammenfassung aller Quellen
    7. Click refresh to generate

    8. Review the summary

    9. Open Studio

    10. Navigate to Studio panel

    11. Select Bericht template

    12. Customize output

    13. Open template menu
    14. Set language and tone

    15. Add extra prompt with focus areas

    16. Generate and export

    17. Click generate button
    18. Review output in Studio-Ergebnisse
    19. Download for external editing

    Result

    Report-style draft you can refine in external tools.

    Workflow 3: Convert Insights into Presentation-Ready Structure

    Goal

    Create a slide deck outline from your research.

    Steps

    1. Prepare source set
    2. Select relevant documents

    3. Ensure key findings are captured

    4. Chat for key points

    5. Ask for main themes
    6. Request bullet points

    7. Save important responses as notes

    8. Use Präsentation template

    9. Open Studio
    10. Select Präsentation template

    11. Set language and style

    12. Generate output

    13. Click generate

    14. Review slide outline

    15. Export and refine

    16. Download the output
    17. Transfer to presentation software
    18. Add visuals and formatting

    Result

    Rapid first draft of slide flow ready for refinement.

    Workflow 4: Manage Long-Running Project Memory

    Goal

    Keep assistant behavior clean and relevant over time.

    Steps

    1. Review memory entries
    2. Open Account page

    3. View stored memory items

    4. Delete outdated entries

    5. Select irrelevant memories
    6. Click delete selected

    7. Or clear all if needed

    8. Keep useful context

    9. Leave entries that help
    10. Remove noise and outdated info

    Result

    Cleaner assistant behavior and reduced noise in responses.

    Workflow 5: Multimodal Analysis

    Goal

    Combine text, images, and audio in one analysis.

    Steps

    1. Upload image
    2. In Chat, click image icon

    3. Select image file

    4. Add audio context

    5. Click audio icon
    6. Record or upload audio

    7. Review transcription

    8. Enter prompt

    9. Ask about both image and audio content

    10. Request specific analysis

    11. Review response

    12. Check image analysis

    13. Verify audio context integration

    14. Save result

    15. Save as note for reference

    Result

    Comprehensive analysis combining multiple input types.

    Workflow 6: Quick Research Synthesis

    Goal

    Rapidly synthesize information from multiple sources.

    Steps

    1. Select diverse sources
    2. Choose documents from different perspectives

    3. Include primary and secondary sources

    4. Generate all-sources summary

    5. Use summary feature

    6. Review key themes

    7. Ask synthesis question

    8. Request comparison or synthesis

    9. Ask for conflicting viewpoints

    10. Generate infographic

    11. Use Infografik template
    12. Visualize key findings

    Result

    Quick synthesis with visual output for sharing.

    Workflow 7: Podcast Script Creation

    Goal

    Create a podcast script from research material.

    Steps

    1. Prepare content
    2. Select source documents

    3. Generate summary for context

    4. Open Studio

    5. Select Podcast template

    6. Set language and tone

    7. Customize for your format

    8. Add extra prompt with:

      • Episode length
      • Target audience
      • Key topics to cover

    9. Generate and review

    10. Create script
    11. Edit as needed

    Result

    Podcast script ready for recording.

    Tips for All Workflows

    Before Starting

    • Clear irrelevant selections
    • Verify API connectivity
    • Check source quality

    During Work

    • Save important outputs immediately
    • Use notes for intermediate results
    • Check citations for accuracy

    After Completion

    • Export final outputs
    • Clean up temporary notes
    • Document your process

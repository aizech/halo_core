# Content Creation Skill

Use this skill when creating written content, blog posts, articles, marketing copy, or any marketing materials.

## When to Use

- Writing blog posts or articles
- Creating marketing copy
- Developing content strategy
- Writing product descriptions
- Crafting social media content
- Producing newsletters

## Content Creation Process

### 1. Understand Purpose

- What is the goal?
- Who is the audience?
- What action should reader take?
- What tone is appropriate?

### 2. Research & Plan

- Understand the topic
- Identify key messages
- Structure the content
- Gather supporting facts

### 3. Write

**Hook/Opening**:
- Attention-grabbing first line
- Clear value proposition
- Promise of benefit

**Body**:
- Logical flow
- Scannable structure
- Supporting examples
- Relevant data

**Call to Action**:
- Clear next step
- Compelling reason to act
- Easy to follow

### 4. Refine

- Edit for clarity
- Check for tone
- Optimize for SEO if applicable
- Ensure consistency

## Content Types

| Type | Length | Key Element |
|------|--------|-------------|
| Blog post | 500-2000 words | Value, structure |
| Article | 1000-3000 words | Depth, sources |
| Product copy | 50-200 words | Benefits, CTAs |
| Social post | 50-280 chars | Hook, engagement |
| Newsletter | 300-800 words | Personal, valuable |

## Best Practices

- Lead with value
- Use clear, simple language
- Tell stories when relevant
- Include specific examples
- Use active voice
- Break up text with headings
- Include visuals descriptions
- End with clear CTAs
- Match brand voice
- Proofread carefully

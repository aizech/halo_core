## Task Coordination

### Delegation Guidelines

| Task Type | Delegate To | Example |
|-----------|-------------|---------|
| Medical questions | medical_team | Symptoms, diagnosis |
| Research | web_researcher | Fact-finding, sources |
| Content | marketing_team | Blog posts, social |
| Data | data_analyst | Numbers, statistics |
| Notes | note_taker | Meeting notes, capture |
| Summary | summarizer | Long documents |

### Escalation Path

1. Simple request → Handle directly
2. Requires expertise → Delegate to specialist
3. Multi-faceted → Assemble team
4. Urgent → Prioritize and notify

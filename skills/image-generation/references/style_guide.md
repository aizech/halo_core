# Image Style Guide

## Photography Styles

| Style | Description | Best For |
|-------|-------------|----------|
| Portrait | Human subjects, professional lighting | Headshots, profiles |
| Landscape | Nature, environments | Travel, nature |
| Product | Clean, controlled lighting | E-commerce |
| Street | Candid, urban | Documentary |
| Macro | Extreme close-up | Details, textures |

## Art Styles

| Style | Characteristics | Use Case |
|-------|----------------|----------|
| Realistic | Accurate, detailed | Technical, medical |
| Impressionistic | Brush strokes, light focus | Artistic, decorative |
| Abstract | Shapes, colors, no defined subject | Modern design |
| Minimalist | Simple, clean, few elements | Tech, lifestyle |
| Vintage | Retro colors, grain | Nostalgic, branding |

## Technical Settings

### Resolution
- Web: 72-150 DPI
- Print: 300 DPI
- Large format: 150-300 DPI

### Aspect Ratios
- Social: 1:1 (square), 4:5 (portrait)
- Video: 16:9
- Photo: 3:2, 4:3

### Color Modes
- RGB: Digital/web
- CMYK: Print

## Lighting Terms
- Key light: Main illumination
- Fill light: Soften shadows
- Back light: Separate subject from background
- Natural: Window/daylight
- Artificial: Studio lights

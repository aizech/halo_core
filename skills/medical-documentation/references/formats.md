# Medical Documentation Formats

## Common Formats

### SOAP Note
```
S - Subjective: Patient's complaints, history
O - Objective: Vital signs, exam findings, labs
A - Assessment: Diagnosis, differential
P - Plan: Treatment, follow-up
```

### Progress Note
```
Date/Time
Provider
Note Type
Content:
- Subjective findings
- Objective findings
- Assessment
- Plan
```

### Discharge Summary
```
Reason for Admission
Hospital Course (day by day)
Procedures Performed
Discharge Diagnosis (primary + secondary)
Medications (current)
Follow-up Appointments
Discharge Instructions
```

### Referral Letter
```
Referring Provider + Date
Patient Demographics
Reason for Referral
Clinical History
Current Medications
Urgency Level
```

## Case Report (CARE Guidelines)
1. Title - Concise, descriptive
2. Abstract - Summary 150-250 words
3. Introduction - Background, why notable
4. Case Presentation - Demographics, history, findings
5. Discussion - Pathophysiology, management, outcomes
6. Conclusion - Key takeaways
7. References

## Documentation Standards
- Date/time for all entries
- Provider identification
- Legible/typed
- Factual, objective
- Complete and timely
- HIPAA compliant

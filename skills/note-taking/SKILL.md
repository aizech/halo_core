# Note-Taking Skill

Use this skill when organizing information, creating structured notes, or capturing key information from content.

## When to Use

- Organizing information from documents/meetings
- Creating structured notes
- Capturing key insights from content
- Building knowledge bases
- Preparing for later reference

## Note-Taking Process

### 1. Capture

Record:

- Main ideas and concepts
- Key facts and data
- Important quotes
- Questions that arise
- Personal observations

### 2. Organize

Structure notes with:

- Clear headings (topic, subtopic)
- Logical grouping
- Consistent formatting
- Numbered/bulleted lists
- Visual hierarchy

### 3. Enhance

Add context:

- Source information
- Date/time relevant
- Connections to other topics
- Personal insights
- Action items if applicable

### 4. Review

Ensure:

- Complete capturing of key points
- Clear and readable
- Searchable structure
- Actionable if needed

## Note Structures

### Cornell Method

```
Topic:
Date:

Main Notes:
[Column for notes]

Summary:
[Column for key points]
```

### Mind Map

```
       [Central Topic]
          /    |    \
    [Sub 1] [Sub 2] [Sub 3]
```

### Outline

```
I. Main Topic
   A. Point 1
      1. Detail
      2. Detail
   B. Point 2
```

## Best Practices

- Write in your own words
- Use abbreviations consistently
- Leave space for additions
- Note sources immediately
- Review and revise soon after
- Keep notes actionable
- Link related concepts

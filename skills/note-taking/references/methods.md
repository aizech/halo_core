# Note-Taking Methods

## Popular Methods

### Cornell Method
```
+------------------+------------------------+
| Cues/Questions  | Notes                 |
| (left column)   | (right column)        |
|                 |                       |
+------------------+------------------------+
| Summary (bottom section)                 |
+----------------------------------------+
```
**Best for:** Lectures, study sessions

### Outline Method
```
I. Main Topic
   A. Subtopic
      1. Detail
      2. Detail
   B. Subtopic
```
**Best for:** Hierarchical information

### Mind Map
```
        [Central Topic]
           /    |    \
      [Branch] [Branch] [Branch]
```
**Best for:** Brainstorming, relationships

### Box Method
```
+-------+-------+
| Box1 | Box2  |
|-------+-------+
| Box3 | Box4  |
+-------+-------+
```
**Best for:** Comparisons, categories

## Best Practices
- Write in your own words
- Use abbreviations consistently
- Leave space for additions
- Note sources immediately
- Review and revise soon
- Keep notes actionable
- Link related concepts

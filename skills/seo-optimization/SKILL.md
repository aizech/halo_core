# SEO Optimization Skill

Use this skill when optimizing content for search engines, researching keywords, or improving web visibility.

## When to Use

- Optimizing blog posts or articles
- Researching keywords
- Improving website content
- Creating SEO strategy
- Analyzing competitor content
- Writing meta descriptions

## SEO Process

### 1. Keyword Research

- Identify primary keyword
- Find related keywords
- Analyze search volume
- Check competition
- Long-tail opportunities

### 2. Content Optimization

**Title**:
- Include primary keyword
- compelling and clickable
- Under 60 characters

**Headings**:
- Use H1 for main title
- H2 for sections
- Include keywords naturally

**Body**:
- Primary keyword in intro
- Scatter keywords naturally
- 1-2% keyword density
- LSI keywords throughout

**Meta**:
- Description under 160 chars
- Include call to action
- Primary keyword

### 3. Technical Factors

- Internal linking
- External authoritative links
- Image alt text
- URL structure
- Page speed
- Mobile friendly

### 4. Measurement

- Track rankings
- Monitor traffic
- Analyze engagement
- Iterate and improve

| Type | Example | Use |
|------|---------|-----|
| Primary | "content marketing" | Main topic |
| Secondary | "content strategy" | Related topics |
| Long-tail | "content marketing for SaaS" | Specific queries |

## Best Practices

- Write for users first
- Use natural language
- Avoid keyword stuffing
- Quality over quantity
- Update content regularly
- Build quality backlinks
- Optimize for featured snippets
- Use schema markup
- Monitor analytics
- Stay updated on algorithm changes

## Common Mistakes

- Keyword stuffing
- Duplicate content
- Ignoring mobile
- Slow page speed
- Thin content
- Missing meta tags
- Poor internal linking

# Web Research Skill

Use this skill when researching topics, verifying facts, or synthesizing information from web sources.

## When to Use

- User asks for information about a topic
- Fact-checking claims or statements
- Finding current/recent information
- Comparing multiple sources
- Synthesizing information from web

## Research Process

### 1. Search Strategy

- Identify key search terms
- Use appropriate sources (news, academic, general)
- Consider multiple search queries
- Look for authoritative sources

### 2. Source Evaluation

Assess each source for:

- **Authority**: Who wrote it? What organization?
- **Currency**: When was it published? Updated?
- **Accuracy**: Is it verifiable? Citations?
- **Purpose**: Informational? Opinion? Ad?
- **Coverage**: What aspects does it cover?

### 3. Information Synthesis

- Extract key facts from multiple sources
- Identify areas of consensus
- Note conflicting information
- Prioritize authoritative sources

### 4. Citation

Cite sources with:

- Title and author
- Publication date
- URL
- Access date (for dynamic content)

## Source Types

| Type | Best For | Considerations |
|------|----------|----------------|
| News | Current events | May lack analysis |
| Academic | Research, data | Peer review |
| Government | Statistics, policy | Authoritative |
| Expert blogs | Deep dives | May have bias |
| Wikipedia | Overview | Check references |

## Best Practices

- Cross-reference multiple sources
- Prioritize primary sources
- Check publication dates
- Note author credentials
- Distinguish facts from opinions
- Acknowledge limitations in search
- Provide balanced coverage
